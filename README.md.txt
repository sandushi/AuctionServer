Connect to the server by using nc in linux or telnet in windows.
Enter the name of the client.Client's name will be registered.
Then enter the symbol which the client needs to bid for.Current prices of the most specific symbols will be shown using a table in another window named "Stock Manager".
Enter the bid for the selected symbol considering the "Stock Manager" table.
A client can bid more than once for a particular symbol.
When the bidding is done client must enter "quit" to exit the bidding by disconneting from server.
